# cgv-springboot-project

## 🖥️ 프로젝트소개
기존 cgv 홈페이지에서 디자인을 바꿔 구현한 풀스택 프로젝트입니다.

## 🕰️ 개발 기간
2024.08 ~ 2024.09

### 🧑‍🤝‍🧑 개발 인원
개인 프로젝트

### ⚙️ 개발 환경
- **Language** : Java 11
- **IDE** : STS4
- **Framework** : SpringBoot 2.6.7
- **Database** : MariaDB 10.6.12
- **ORM** : Mybatis 2.2.0
